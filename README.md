## User List
Tem como objetivo final cadastrar e fazer login de usuários, que ao logar é retornado uma listagem de users cadastrados. 
Tendo controle de token para cada user Logado.

Techs utilizadas:

## BackEnd
- NodeJS
- Prisma
- MongoDB
- Express
- JWT - Jason Web Token

## FrontEnd
- Vite
- ReactJS
- TailWindCSS
- JWT - Jason Web Token
- Axios
